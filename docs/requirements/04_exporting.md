# Exporting Functionality

## 1. Access Levels
*   **Free**: JSON, XML formats.
*   **Pro/Team**: Java, Python, JavaScript, TypeScript, JSON, XML.

## 2. Logic & Naming
*   **Variable Names**: Use "Saved Name" from UI. Prompt user if missing. Editable in preview.
*   **File Naming**: Suggest filenames based on POM Page names (e.g., `LoginPage.java`).

## 3. Structure
*   **Normal Mode**: Simple variable snippets.
*   **POM Mode**: Full class structure with framework-specific annotations (e.g., `@FindBy`).
*   **Multi-File**: Support for exporting multiple POM pages as individual files at once.
